# Tasker

Tasker is a simple task management app built with React.

## Features List

- [x] Have a Header
- [x] Have a Footer
- [x] Have a Hero Section
- [x] List of Tasks
- [x] Delete a Task
- [x] Add a Task
- [x] Delete All Tasks
- [x] Edit a Task
- [x] Mark a Task Favorite/Unfavorite
- [x] Search Tasks
- [x] Show No Tasks Found
